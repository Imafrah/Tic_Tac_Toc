Tic Tac Toe ❌⭕
A web-based version of the classic Tic Tac Toe game where two players can play on the same device.

**Features:**
- Two-player mode.
- Highlights the winner.
- Option to reset and start a new game.
- Clean UI and grid layout.

**Tech Stack:**
- HTML
- CSS
- JavaScript
